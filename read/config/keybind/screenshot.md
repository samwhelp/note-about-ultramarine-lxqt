# 螢幕截圖

* [設定片段](https://github.com/samwhelp/ultramarine-lxqt-adjustment/tree/main/prototype/main/lxqt-config/Main/asset/overlay/etc/skel/.config/openbox/helper/share/gen/openbox-gen-rc/Section/Keybind/Screenshot.php#L3-L14)

| 按鍵組合        | 功能                  | 執行指令                   |
| --------------- | --------------------- | -------------------------- |
| `Print`         | 整個桌面螢幕截圖      | `screengrab --fullscreen`  |
| `Alt + Print`   | 啟動螢幕截圖程式      | `screengrab`               |
| `Win + Print`   | 目前視窗截圖          | `screengrab --active`      |
| `Ctrl + Print`  | 選取螢幕畫面區塊截圖  | `screengrab --region`      |
